# Live Auction Engine

## Overview

A real-time auction system where users compete by placing bids before the timer expires.

---

## Features

- Live auction countdown
- Dynamic bid updates
- Highest bid tracking
- Automatic winner selection
- Event-driven workflow

---

## Files

- auction.py
- engine.py

---

## Usage

```bash
python engine.py
```

---
